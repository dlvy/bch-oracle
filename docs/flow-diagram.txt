KEY COMPONENTS:
===============

1. HTTP API (router.gleam)
   - Handles REST requests
   - Creates/manages tasks
   - Triggers deployments

2. Database (db.gleam)
   - SQLite persistence
   - Stores tasks and results
   - Provides CRUD operations

3. Scheduler (worker.gleam)
   - OTP process per task
   - Runs on interval
   - Isolated failures

4. LLM Client (llm.gleam)
   - Fetches URL content
   - Calls OpenAI API
   - Returns response

5. Parser (parser.gleam)
   - Extracts values
   - Four strategies
   - Error handling

6. BCH Bridge (bch.gleam)
   - Spawns Node.js
   - Publishes on-chain
   - Parses responses

7. Smart Contract (oracle.cash)
   - Verifies signatures
   - Creates OP_RETURN
   - Stores results


DATA FORMATS:
=============

Task JSON:
{
  "id": 1,
  "name": "BCH > $500?",
  "source_url": "https://...",
  "prompt": "Is BCH above $500? {content}",

  "interval_seconds": 3600,
